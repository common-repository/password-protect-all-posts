== Description ==

This plugin puts a global password selected by you on all posts. Based on Matt Mullenwegs plugin "Protect old posts"

I've looked for a plugin like this but couldn't find one that fitted my needs so i this one. I hope it will be helpfull for someone else to.

**BEWARE: This will write over all old passwords and replace it with the one you select.**

== Installation ==

1. Upload `plugin-name.php` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Select a global password for all posts under 'Password Protect All Posts' under settings.
